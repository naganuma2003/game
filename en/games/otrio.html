OTRIO — Tic-Tac-Toe with 3 Sizes
Players:
Mode:
CPU Level:
Evaluation
Win conditions: Using your color, achieve any of the following: (1) 3 of the same size in a row/column/diagonal, (2) Small-Medium-Large ascending in a row/column/diagonal, (3) All 3 sizes (concentric rings) in one cell.
Pieces: 3 of each size (9 total). You can place on any empty size slot.
