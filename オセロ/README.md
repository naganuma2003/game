# オセロ / Reversi

ブラウザで遊べるオセロ。HTML / CSS / JavaScript のみ（ライブラリ不要）。

## 遊び方

`index.html` をブラウザで開くと、まず **対戦設定画面** が表示されます。黒（先手）・白（後手）それぞれを「人間」か「CPU」から選び、CPU なら強さ（弱い／強い）を指定して「対戦開始」。緑の盤上のヒント（白い点）をクリックして石を置きます。

## 機能

- **対戦設定画面** … 黒・白を個別に「人間／CPU」に設定。CPU は強さも個別指定
  - 人間 vs CPU、人間 vs 人間、CPU vs CPU（観戦）すべて可能
- CPU の強さは **5段階** から選択
  - **Lv1 入門** … 合法手からランダム
  - **Lv2 やさしい** … 最も多く取れる手を選ぶだけ（先読みなし）
  - **Lv3 ふつう** … 評価関数 + αβ法（浅め・中盤2手読み）
  - **Lv4 強い** … αβ法（中盤4手・終盤6手読み）
  - **Lv5 最強** … αβ法（中盤5手・終盤8手読み）
- 合法手のハイライト表示と、置いた石の反転アニメーション
- **一手戻る** … 直前の局面に戻す（対 CPU では自分の手＋CPU の応手をまとめて戻し、自分の番に復帰）
- **AI アシスタント** … ON にすると自分の番に、打てる各マスへ AI の評価値を表示。どの CPU よりも深く読み（中盤7手＋着手順序付きαβ法、残り12マス以下は**最後まで完全読み切り**）、最善手を選び続ければ相手に勝てる強さ。数値が大きいほど有利で、上位を色分け（**1位=赤・2位=オレンジ・3位=黄色**、同点は同色）
- 打てる手がない場合は「パス」、両者打てなくなったら自動で終局・勝敗判定
- ゲーム中はいつでも「設定を変える」で対戦設定画面に戻れる

## ファイル構成

```
ゲーム/
├── index.html      画面
├── css/style.css   スタイル
└── js/
    ├── ai.js       盤面ロジック・合法手生成・CPU 思考
    └── game.js     UI とゲーム進行の制御
```

## ローカルサーバーで動かす（任意）

```
python -m http.server 8754
```

ブラウザで http://localhost:8754 を開く。
