# カルカソンヌ（Web版 v1）

ブラウザだけで動く、タイル配置ボードゲーム「カルカソンヌ」の対CPU/ホットシート対応版です。ビルド不要。

## 遊び方
`index.html` をブラウザで開いて設定後「対局開始」を押すだけ。

- 引いたタイルを、既存タイルと**辺の地形（道・都市・草原）が一致**するように置きます。
- **回転**ボタン（または R キー）でタイルを回し、盤上で**青く光るマス**をクリックして配置。
- 配置したタイルに、自分の**ミープル**（各自7個）を置けます。
  - 都市 → 騎士 / 道 → 盗賊 / 修道院 → 修道士 / **草原 → 農夫（寝そべって配置）**
  - すでに誰かのミープルが乗っている地形（草原含む）には置けません。
- 道・都市・修道院が**完成**するとミープルが手元に戻り、その地形の**最多保有者**が得点します（同数なら全員）。
- **農夫は完成という概念がなく、ゲーム終了まで戻りません。** 終了時、その草原に接する**完成した都市**を得点化します。
- 盤面はドラッグで移動、マウスホイールで拡大縮小できます。

## 色の見方
- **オレンジ（城壁・胸壁つき）= 都市**
- **緑 = 草原**
- 灰色の線 = 道、中央の家 = 修道院、青い盾 = 紋章（都市の得点+α）

## 得点
| 地形 | 完成時 | ゲーム終了時（未完成） |
|------|--------|------------------------|
| 都市 | 2点 × タイル数（＋紋章ごとに2点） | 1点 × タイル数（＋紋章ごとに1点） |
| 道   | 1点 × タイル数 | 1点 × タイル数 |
| 修道院 | 9点（周囲8マスすべて埋まる） | 1点 ＋ 周囲の埋まったマス数 |
| 草原（農夫） | —（終了時のみ） | その草原に接する**完成した都市1つにつき3点** |

山札（71枚＋開始タイル）が尽きるとゲーム終了。未完成の道・都市・修道院は縮小得点、農夫は上表どおりに計算し、最高得点者が勝ちです。

## 設定
- **人数**: 2〜5人
- **各プレイヤー**: 人間 / CPU を席ごとに選択（全員CPUの観戦も可）
- **CPUの強さ**: やさしい / ふつう
- **CPUの速度**: 速い / ふつう / ゆっくり

## 実装メモ
- 基本セット72枚のタイル分布（A〜X）を実装。開始タイルは都市＋直線道のタイル（D）。
- 地形の連結は Union-Find で管理し、盤をまたいだ都市・道を1つの地形として判定。
- **農夫（草原）ルールに対応**。草原はタイル周囲を8つのハーフ辺に分割して連結判定し、終了時に接する完成都市を得点化します。
