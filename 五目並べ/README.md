# 五目並べ / Gomoku

ブラウザで遊べる五目並べ。HTML / CSS / JavaScript のみ（ライブラリ不要）。

## 遊び方

`index.html` を開くと対戦設定画面が出ます。黒（先手）・白（後手）をそれぞれ「人間」か「CPU」に設定し（CPU は強さも選択）、「対戦開始」。15路盤の交点をクリックして石を置き、**先にタテ・ヨコ・ナナメいずれかに 5 つ並べたら勝ち**です。

## 機能

- 15×15 路盤・勝敗判定（5連）・引き分け（盤面満杯）
- 対戦相手を黒/白それぞれ人間／CPUに設定
- CPU の強さ 4 段階
  - **Lv1 入門** … ばらつきのある簡易評価
  - **Lv2 やさしい** … 勝ち/受けを見る基本評価
  - **Lv3 ふつう** … 攻防の形を評価
  - **Lv4 強い** … 1手先読み（相手の最大の脅威も考慮）
- **AIアシスタント** … ON で自分の番に各候補手の評価値を表示（上位を赤/オレンジ/黄で色分け、必勝・必受は★）
- 一手戻る（対 CPU では自分の手＋CPU の応手をまとめて戻す）
- 最後に置いた石に赤いマーカー

## ファイル構成

```
五目並べ/
├── index.html
├── css/style.css
└── js/
    ├── ai.js     盤面ルール・形の評価・CPU 思考・アシスタント評価
    └── game.js   UI・対局進行
```

## ローカルサーバー（任意）

```
python -m http.server 8756
```
