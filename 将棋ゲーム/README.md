# 将棋 / Shogi

ブラウザで遊べる本将棋。HTML / CSS / JavaScript のみ（ライブラリ不要）。

## 遊び方

`index.html` を開くと対戦設定画面が出ます。先手（下）・後手（上）をそれぞれ「人間」か「CPU」に設定し（CPU は強さも選択）、「対戦開始」。

- 自分の駒をクリック → 緑のマーカーが合法手。行き先をクリックで移動
- 敵陣（3段目以内）に関わる移動で成れる場合は「成る／成らない」を選択（成り強制の場合は自動）
- 取った駒は持ち駒に。**持ち駒をクリック → 打てるマスをクリック**で打てる
- 王手は上部に表示、詰みで対局終了

## 実装しているルール

- 8 種の駒の動き・成り（と金/成香/成桂/成銀/竜/馬）
- 持ち駒の打ち込み、**二歩**・行き所のない駒（最奥段の歩香・最奥2段の桂）の禁止
- 王手・詰みの判定、自玉が王手のままになる手の禁止
- 一手戻る（対 CPU では自分の手＋CPU の応手をまとめて戻す）
- **AI アシスタント** … ON にすると自分の番に推奨手を盤上にハイライト＋表示（緑枠=移動元・赤枠=移動先）。駒や持ち駒を選ぶと、各行き先に評価値を表示し上位を色分け（1位=赤・2位=オレンジ・3位=黄）

> 簡略化: **打ち歩詰め**の禁止のみ未実装です。

## CPU の強さ

材料評価 + αβ探索（着手順序付け・読みノード数の上限あり）。

- **弱い** … 1手読み（取り合い重視・ばらつきあり）
- **普通** … 2手読み
- **強い** … 3手読み（局面により少し時間がかかります）
