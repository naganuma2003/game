# リーチ麻雀（Web版 v1）

ブラウザだけで動く、日本式リーチ麻雀の対CPU（3人）対戦ゲームです。ビルド不要。

## 遊び方
`index.html` をブラウザで開いて「対局開始」を押すだけ。
- 自分の手牌をクリックして打牌します（右に離れているのがツモ牌）。
- テンパイ時に「リーチ」、和了形になると「ツモ」「ロン」ボタンが出ます。
- 東風戦（東1〜東4局、親は連荘）。持ち点25000スタート。

## 実装済み
- 牌山・配牌・ツモ・打牌（4人打ち）
- 東風戦 / 半荘戦の選択
- 各席を CPU / 人間 で選択（全CPUの観戦も可）
- CPUの強さ5段階・速度4段階を選択
- 対局ログ（左上・既定は折りたたみ／リーチ・鳴き・和了・流局を記録）
- 和了後10秒で次局を自動開始（「次へ」で即時も可）
- Mリーグ準拠寄り：喰いタンあり・頭ハネ・トビなし・親の和了/テンパイで連荘
- 鳴き：ポン・チー・大明槓・暗槓・加槓（嶺上開花・カンドラ・カン裏）
- 和了判定：通常形 / 七対子 / 国士無双（鳴きありは通常形）
- リーチ / 一発 / ツモ / ロン / フリテン / 海底・河底
- 役判定・符・翻・点数計算（鳴きの喰い下がり対応、満貫〜役満、数え役満）
- ドラ・裏ドラ・カンドラ・赤ドラ（赤5を各色1枚）
- 最終結果に順位点（ウマ +30/+10/−10/−30、原点25000）を表示
- 途中流局：九種九牌・四風連打・四家立直（親は連荘）
- CPU：牌効率ベースAI。役牌ポンに加え、役確定後は手が進む鳴き（チー/ポン）と暗槓も判断（赤5は残す）。テンパイで自動リーチ
- 牌はCSS描画（絵文字不使用で軽量・見やすい。萬=黒/筒=青/索=緑、赤5は赤表示）
- 流局・テンパイ料・本場・供託（リーチ棒）

## ファイル構成
`index.html` は **CSS・JS をすべて埋め込んだ単一ファイル**です。これ単体で動くので、ダブルクリックで起動できます（外部ファイル不要）。

- `index.html` … 配布用の単一ファイル（これだけで動作）
- `css/style.css` … 元になったスタイル（参考用ソース）
- `js/tiles.js` … 牌の定義とユーティリティ（参考用ソース）
- `js/agari.js` … 和了形の判定・分解、テンパイ/待ち（参考用ソース）
- `js/score.js` … 役・符・点数計算（参考用ソース）
- `js/ai.js` … CPU思考（シャンテン計算）（参考用ソース）
- `js/game.js` … ゲーム進行とUI（参考用ソース）

> `css/` `js/` は `index.html` に埋め込み済みの元ソースです。ゲーム実行には使われません（編集の参考用）。
